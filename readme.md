## Changelog

- [x] 支持 Windows & Mac 版本
- [x] 使用 ⇧ + ⌘ + V 呼出窗口，再按一次可隐藏
- [x] 双击相应卡片进行复制
- [x] 支持日历配置
- [x] 实现剪切板功能
- [x] 支持系统快捷键
- [ ] 日历支持创建待办，关联系统日历
